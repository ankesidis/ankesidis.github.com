Mobile Friendly Without CSS Framework

Mr. Chatzis came with a very old Joomla website with a non mobile friendly template (without any css framework) and a lot of custom code made on it (lots of iframes!) and he ask for my help, making a mobile friendly version of it.

The main problem was that, because of the iframes, I was not be able to use just one .css files to make achive the mobile friendly view. So I've decide to go for one custom style for the template and inline styles for any external iframes. Also some custom javascript actions for mobile view (mobile menu).
